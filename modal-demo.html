🔧 使用方法

只需在HTML中引入一个JS文件：

⚙️ 自定义配置

编辑JS文件中的配置项：

const config = { qrImageUrl: 'https://qm.qq.com/q/Q4YTVK5T6q', workTime: '工作时间：7×24小时在线服务', buttonColor: '#52c41a', buttonIcon: '💬' };

🎮 JavaScript API

通过代码控制弹窗：

// 显示弹窗 CustomerService.show(); // 关闭弹窗 CustomerService.hide(); // 更新配置 CustomerService.updateConfig({ qrImageUrl: '新的QQ客服链接', workTime: '新的工作时间' });

❓ 常见问题

Q: 如何更换QQ客服链接？

A: 修改 config.qrImageUrl 为你的QQ客服链接

Q: 二维码显示不出来怎么办？

A: 可以点击"直接打开QQ"按钮，或检查网络连接

Q: 如何更改按钮位置？

A: 修改CSS中的 bottom 和 right 属性

Q: 可以改成其他颜色吗？

A: 可以！搜索CSS中的 #52c41a 替换为你喜欢的颜色
