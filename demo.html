🎯 QQ在线客服插件演示

这是一个纯JavaScript实现的QQ在线客服悬浮窗，无需依赖jQuery，开箱即用！

✨ 主要特性

  • 纯原生JavaScript，零依赖
  • 自动悬浮在页面右下角
  • 支持多个QQ客服账号
  • 显示客服电话
  • 现代化渐变UI设计
  • 平滑过渡动画效果
  • 支持悬停/点击两种交互模式
  • 响应式设计，兼容移动端

🚀 使用方法

只需在HTML中引入一个JS文件即可：

<script src="qq-customer-service.js"></script>

⚙️ 自定义配置

如需自定义，请编辑 qq-customer-service.js 文件中的配置项：

const config = { qqs: [ { name: '售前客服', qq: '123456789' }, // 改成你的QQ号 { name: '售后客服', qq: '987654321' } ], tel: '400-123-4567', // 客服电话 position: { bottom: '80px', // 距离底部 right: '20px' // 距离右侧 }, autoShow: true, // 是否自动展开 hoverToShow: true // true=悬停展开, false=点击展开 };
💡 提示： 请查看页面右下角，客服悬浮窗已经自动加载！鼠标移上去试试看。

📝 常见问题

Q: 如何修改QQ号？

A: 编辑 qq-customer-service.js 文件中的 qqs 数组，将 qq 字段改为你的QQ号。

Q: 如何改变位置？

A: 修改 config.position 中的 bottom 和 right 值。

Q: 如何改为点击展开？

A: 将 hoverToShow 设置为 false。
